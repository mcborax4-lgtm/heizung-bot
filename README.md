# Germany Heizung Telegram Bot

Telegram-бот для автоматической обработки запросов клиентов в сфере отопительных систем.  
Бот поддерживает **русский и немецкий язык**, определяет язык сообщения пользователя и отвечает на основе базы знаний.

Проект создан как демонстрационный прототип для автоматизации первичной клиентской поддержки.

---

# Функционал

- Автоматическое определение языка сообщения (RU / DE)
- Ответы на основе базы знаний
- Категоризация запросов по ключевым словам
- Логирование сообщений пользователей
- Система статистики пользователей
- Админ-команда `/stats`
- Reply-клавиатура с быстрыми действиями

# Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/username/repository-name.git
cd repository-name
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Создать .env файл

```BOT_TOKEN=your_telegram_bot_token```

### 4. Запустить бота

```bash
python bot.py
```

## Структура проекта
```
bot.py              # основной файл бота
heizung_bot.log     # логирование сообщений
.env                # токен бота (не добавляется в git)
```

Архитектура
KNOWLEDGE_BASE

База знаний, содержащая категории вопросов и варианты ответов.

detect_language()

Определяет язык сообщения пользователя.

get_response()

Подбирает ответ из базы знаний на основе ключевых слов.

handle_message()

Основная функция обработки сообщений пользователей.

### Пример использования

Пользователь пишет сообщение:
```Heizung funktioniert nicht```

Бот автоматически определяет язык и отвечает:
```Bei Heizungsproblemen: Ist die Heizung komplett kalt oder wird sie nicht richtig warm?```

### Логирование

Все сообщения пользователей сохраняются в файл:

```heizung_bot.log```

Это позволяет анализировать обращения пользователей и улучшать базу знаний.
